# Case Library (Design Benchmarks)

Use these as style and structure references. Do not copy blindly; extract patterns.

## A) Minimal product style (clean, premium, restrained)
- OpenAI: https://openai.com/
  - Learn: large value-proposition headline, calm spacing, clear CTA hierarchy.
- Claude: https://claude.ai/
  - Learn: minimal palette, concise copy blocks, low-noise layout.
- Apple: https://www.apple.com/
  - Learn: typography-led storytelling, disciplined spacing, premium imagery.

## B) Tourism / destination information style
- Japan Official Travel (JNTO): https://www.japan.travel/en/us/
  - Learn: destination discovery structure, visual-led browsing, practical travel info grouping.
- Visit Beijing (official city tourism): https://english.visitbeijing.com.cn/
  - Learn: city content categorization, attraction/event/travel-support sections.
- Visit China (gov portal): https://english.www.gov.cn/services/visitchina
  - Learn: policy + traveler guidance blending, high-clarity info blocks.
