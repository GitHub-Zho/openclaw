# UX Principles (Practical)

## Typography hierarchy
- H1: 2.5x body (or clamp-based equivalent)
- H2: 1.7-2.0x body
- H3: 1.3-1.5x body
- Body: 16px baseline
- Label/meta: 12-14px
- Keep heading weight consistent (e.g., 650-700).

## Spacing rhythm
- Use 8pt grid tokens: 4/8/12/16/20/24/32/40.
- Keep section gaps larger than internal card gaps.
- Never mix arbitrary spacing numbers unless necessary.

## Minimalist visual language
- One neutral base + one accent.
- 1px borders over heavy shadows.
- One radius scale for all cards/buttons.
- Remove decorative noise unless it improves comprehension.

## Image system
- Use one visual family per page (color temp + contrast).
- Normalize with consistent filters if sources vary.
- Place images to support content, not distract from CTA.

## Interaction clarity
- Primary CTA: one clear action.
- Secondary CTA: non-competing style.
- Keep nav labels literal and short.
- FAQ should answer purchase/signup objections directly.
