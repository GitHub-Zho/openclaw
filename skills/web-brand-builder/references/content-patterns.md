# Content Patterns for Program/Trip Websites

## Home page pattern
1. Hero: one sentence value proposition
2. Program card: route + duration + price anchor
3. Why join: 3-5 short benefits
4. Preview CTA: view detailed program / handbook
5. Contact minimal block

## Program detail page pattern
1. Intro summary + package basics
2. Visual strip (3 themed images)
3. Why this route
4. Day-by-day cards
5. Included/Excluded
6. FAQ
7. Register block

## Handbook page pattern
1. Purpose and context
2. City highlights
3. Accommodation/support
4. Dietary and safety notes
5. Application entry

## Writing rules
- Use short paragraphs (2-4 lines).
- Prefer concrete nouns and numbers.
- Remove internal-tool references in user-facing copy.
- Repeat key pricing constraints where users decide.
