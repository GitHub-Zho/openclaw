# Content Patterns for Program / Trip Websites

## Home page
1. Hero: one-sentence value proposition + primary CTA
2. Program preview: route + duration + price anchor
3. Why join: 3-5 concrete benefits
4. Handbook/detail preview CTA
5. Minimal contact block

## Program detail page
1. Intro summary + package basics
2. Visual strip (consistent style)
3. Why this route
4. Day-by-day structure
5. Included / excluded
6. FAQ
7. Register block

## Handbook page (internal)
1. Purpose and context
2. City/program highlights
3. Accommodation/support notes
4. Dietary/safety notes
5. Application entry

## Writing rules
- Prefer short paragraphs (2-4 lines).
- Prefer concrete nouns, numbers, and constraints.
- Repeat key decision info near CTA (price, inclusion limits, deadlines).
- Remove internal-tool references in user-facing copy.
- Use plain language over marketing jargon.
